# Progressbar95 ModLoader Project
A tiny, modular modloader for Progressbar95 intended to easily create mods without messing up with game files

![Logo](https://raw.githubusercontent.com/ryzhpolsos/pbml/refs/heads/main/logo.png)

## Installation
0. Download the archive from [releases page](https://github.com/ryzhpolsos/pbml/releases)

### Windows
1. Make sure you have Java and Python installed
2. Extract the archive
3. Launch `install.bat` as administrator
4. When loader asks for game directory path, enter it
5. Wait for patch process to complete
6. Launch game and enjoy

### Linux
1. Make sure you have Java and Python installed
2. Extract the archive
3. Navigate to the extracted files location in terminal and type `bash install.sh`
4. When loader asks for game directory path, enter it
5. Wait for patch process to complete
6. Launch game and enjoy

## Mod installation
1. Download mod
2. Extract it to `[game dir]/Resources/mods` directory

## Mod creation
1. Create a subdirectory inside of `[game dir]/Resources/mods`
2. Put a file called `main.lua` inside of it
3. Refer to [engine api](https://docs.coronalabs.com/api/index.html) and [pbml documentation](https://ryzhpolsos.github.io/pbml/docs)
